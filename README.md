# House-Price-Prediction-System

A modern **Machine Learning** web application that predicts house prices based on property area using a **Linear Regression** model. The project integrates **FastAPI** for the backend and **Streamlit** for an interactive user interface, delivering fast and accurate real-time predictions.

##  Features

* 🏠 Real-time House Price Prediction
* 🤖 Linear Regression Model
* ⚡ FastAPI REST API
* 🎨 Modern Streamlit Dashboard
* 📊 Data Preprocessing & Model Training
* 💾 Model Saving with Joblib
* 🛡️ Input Validation & Error Handling

##  Tech Stack

* Python
* Pandas
* Scikit-learn
* FastAPI
* Streamlit
* Joblib
* Uvicorn

##  Run the Project

```bash
python train.py
uvicorn main:app --reload
streamlit run app.py
```
